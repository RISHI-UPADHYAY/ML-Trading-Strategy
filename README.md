# ML Trading Strategy (Alpha Discovery)

## Overview
This project builds a machine learning-based trading strategy to detect weak predictive signals in noisy financial data.

## Key Features
- Feature engineering using returns, momentum, volatility, and mean reversion
- Logistic regression model for prediction
- Confidence-based trading strategy
- Transaction cost modeling
- Backtesting with Sharpe ratio and drawdown evaluation

## Key Insights
- Financial markets are highly noisy
- Accuracy is not a reliable metric for trading
- Feature engineering and strategy design are critical

## Results
- Strategy evaluated using Sharpe ratio and drawdown
- Demonstrates importance of filtering and risk management

## Tech Stack
- Python, Pandas, NumPy, Scikit-learn, Matplotlib
